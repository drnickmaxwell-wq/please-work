# 1. Global Champagne Design Principles (Loaded Ruleset)

All sections and components in this Atlas are evaluated against the following immutable rules:

- **Gradient Law:**  
  `linear-gradient(135deg, magenta → teal → soft gold)`  
  Gold ≤ 4% area coverage.

- **Layer Stack:**  
  Gradient → Wave Mask → Particles → Grain → Glass → Content

- **Opacity Rules:**  
  Particles ≤ 0.08  
  Grain ≤ 0.06  
  Glass blur ≈ 18–24px  
  Gold accents ≤ 1–2% of the section

- **Typography:**  
  Playfair Display (serif) = luxury headings  
  Inter (sans) = body & UI  
  Montserrat/Lora = deprecated except for legacy reference

- **Motion:**  
  Parallax ≤ 6px  
  Hover ≤ 1.03  
  All animations PRM-safe

- **Spacing Rhythm:**  
  Composite Bonding Polished Page (Phase 1B) is the master spacing reference.
