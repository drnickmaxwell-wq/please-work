# Champagne Ecosystem — Manus Design Atlas v1

## 0. Purpose
This Atlas describes the complete design landscape created by Manus AI and
refined under the Champagne Design System for St Mary’s House Dental.

It unifies:
- Page-by-page structure
- Component archetypes
- Effects library (waves, particles, grain, glass)
- Status of each major section (ready / needs re-skin / deprecated)
- Gaps where new design is required.

## 1. Core Champagne Rules
- Gradient law: 135°, magenta → teal → soft gold, ≤ 4% gold.
- Layer stack: Gradient → Wave → Particles → Grain → Glass → Content.
- Type: Playfair Display (headings) + Inter (UI/body).
- Motion: subtle, PRM-safe, max parallax 6px, max hover scale 1.03.
