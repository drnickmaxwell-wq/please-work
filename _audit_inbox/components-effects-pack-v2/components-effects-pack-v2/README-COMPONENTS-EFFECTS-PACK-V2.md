# Components & Effects Pack V2 (Consistent Names)

This pack contains **exact filenames** used by our pages and scaffolds:

- `components/ui/Button.tsx`
- `components/effects/CoastalWaves.tsx`
- `components/effects/GlowSpotlight.tsx`
- `components/effects/Reveal.tsx`

## Install

Just copy the folders into the **root of your repo** where `package.json` lives.

Resulting structure:
```
/components
  /ui
    Button.tsx
  /effects
    CoastalWaves.tsx
    GlowSpotlight.tsx
    Reveal.tsx
```

No other changes required. The existing pages already import these names.
