# Hero Gilded Preview

This preview isolates the gilded Champagne hero so the motion polish can be evaluated without affecting the production homepage.

## Upcoming smoothing flag

PR 3 will introduce a motion smoothing flag for the parallax tuning. To toggle it once available, adjust the `data-smoothing` attribute on the `<ChampagneHeroGilded />` root element or set the `--parallax-smoothing` custom property on `.preview-hero-gilded` within this route.
